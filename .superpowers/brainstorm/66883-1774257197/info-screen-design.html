Comportamento update (opzione A)

Clic "Aggiorna" → scarica DMG da GitHub → scrive script bash in /tmp → lancia script → l'app si chiude → lo script aspetta, monta DMG, copia il .app, smonta, rilancia la nuova versione.

Il check versione avviene automaticamente all'avvio del tab (una volta per sessione) + manualmente con "Controlla aggiornamenti".
